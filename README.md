# Nexus Analytics Dashboard

A modern SaaS Market Analytics Dashboard built with React, TypeScript, Vite, and Tailwind CSS.

## Overview

Nexus Analytics Dashboard is a real-time analytics platform designed to visualize and manage multiple data sources including:

- Stock Market Analytics
- Cryptocurrency Analytics
- Weather Analytics
- Sports Analytics
- API Monitoring & Management
- Reports & CSV Export
- Notification Center
- User Settings & Preferences

The dashboard features a modern dark theme, interactive charts, responsive layouts, reusable components, and scalable architecture.
